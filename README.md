# EPP — Guia de Referência

Guia completo de execução do ciclo de controle interno do pagamento de pessoal militar, desenvolvido para facilitar a execução pelas equipes escaladas.

## Sobre

O **Exame de Pagamento de Pessoal (EPP)** é uma atividade de controle interno obrigatória e mensal em todas as Organizações Militares (OM), com o objetivo de verificar a correção dos pagamentos realizados pelo SIPPES / CPEx.

Este guia reúne, em formato interativo, todo o conteúdo necessário para a execução do EPP:

- Ciclo completo com os 15 passos (Anexo A — EB90-N-02.001)
- Base legal comentada e categorizada
- Glossário de siglas com filtro e ordenação
- Equipe EPP: composição e restrições
- Controle de frequência
- Modelo do Relatório EPP (Anexo B — EB90-N-02.001)
